# 👨‍💼 Employee Attrition Prediction Using Machine Learning

## 📌 Project Overview
This project predicts whether an employee will leave the company or stay using Machine Learning techniques. It helps organizations understand employee behavior and reduce attrition rate.

---

## 🎯 Objective
To build a machine learning model that can classify employees as:
- Will Stay
- Will Leave

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- VS Code

---

## 📊 Dataset Information
The dataset contains employee-related features such as:
- Age
- Department
- Job Role
- Salary
- Experience
- Job Satisfaction
- Attrition (Target variable)

---

## 🤖 Machine Learning Model
- Algorithm: Random Forest Classifier
- Steps:
  - Data Cleaning
  - Handling Missing Values
  - Encoding Categorical Data
  - Feature Selection
  - Model Training
  - Model Evaluation

---

## 📈 Model Evaluation
- Accuracy Score
- Confusion Matrix
- Classification Report
